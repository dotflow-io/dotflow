# Visão geral

Conceitos introdutórios do **Dotflow**—como fluxos de trabalho, tarefas e execução se encaixam—antes dos guias práticos.

Nesta secção: **fluxo de trabalho e tarefas**, **contexto**, **provedores** (storage, notify, log, scheduler), **status e ciclo de vida**, **overlap no cron** e **modos de processamento** (sequencial, segundo plano, paralelo). Cada página aponta para tutoriais e referência de API.
